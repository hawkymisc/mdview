# 概要

このファイルは日本語のファイル名 (`概要.md`) でも mdview が正しく扱えることを確認するためのサンプルです。

## URL エンコーディング

ファイル名は URL 上では `encodeURI` でエンコードされますが、表示は元の日本語のまま行われます。

## TOC のアンカー

英数を含まない見出しの ID は `encodeURIComponent` のフォールバックが使われます。たとえば「概要」は URL fragment として `#%E6%A6%82%E8%A6%81` のように動作します。

### 詳細

サイドバーの目次から「概要」リンクをクリックして、URL hash がブラウザ上で正しく動作することを確認できます。
